# TianChi_Unet
  2017广东政务数据创新大赛——智能算法赛

- 智能算法赛以“土地智能监管”为主题，使用2015年和2017年分别获取到的广东省某地的卫星图片，识别出两年之间新增的人工地上建筑物（不包括道路）所占的像元图斑。
